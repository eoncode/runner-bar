05 — The product

Run-bar is the missing piece

Three primitives, one popover. Built native in Swift 6.2 with Liquid Glass for macOS 26.

01

Workflow status

Live runs, jobs, and steps across every repo and org. Re-run, cancel, copy logs — no browser, no terminal.

02

Local runner manager

Auto-discovers runners on this Mac. Add, start, stop, deregister — tokens and install commands handled for you.

03

AI failure hooks

On failure, fire a shell command in Terminal. $FAILURE_LOG + $BRANCH inject context. Claude, Codex, Gemini, Aider — any CLI.

06 — Aesthetics & core value

A native Mac surface for CI

Built on Apple's Liquid Glass. Designed to disappear when you're shipping — and surface the moment something breaks.

Translucent depth

Liquid Glass surfaces, aurora gradients, depth-aware shadows. Looks like the OS, not a webview.

Zero ceremony

Install with one curl. Picks up your existing GitHub token. No Apple Developer account required.

Private by default

No analytics, no telemetry. Talks only to api.github.com. Tokens stay in Keychain.
