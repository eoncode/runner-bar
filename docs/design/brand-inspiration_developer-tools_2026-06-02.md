# Brand Inspiration — RunnerBar
**Category:** macOS developer tooling · GitHub Actions / self-hosted runners · AI-augmented dev workflow
**Date:** 2026-06-02
**Subject repo:** [eoncode/runner-bar](https://github.com/eoncode/runner-bar) — "GitHub Actions, local runners, and AI failure recovery — in your macOS menu bar." (Swift 6.2, macOS 26+, Liquid Glass)
**Focus areas:** Brand identity · Visual design

---

## 1 · Where RunnerBar sits

RunnerBar lives at the intersection of three brand categories. Each pulls its visual language from a different lineage — and each shows up below in the swipe file.

| Category | What it signals | Reference brands |
|---|---|---|
| **macOS-native menu bar utility** | Apple-like polish, "Made for Mac" trust, system-level integration | [CleanShot X](https://cleanshot.com), [Raycast](https://www.raycast.com) |
| **GitHub Actions / CI runner** | Speed, cost, drop-in replacement, technical credibility | [Blacksmith](https://www.blacksmith.sh), [Depot](https://depot.dev) |
| **AI-era developer tool** | Agentic, modern, "born for the AI era" | [Cursor](https://cursor.com), [Warp](https://www.warp.dev), [Linear](https://linear.app), [Zed](https://zed.dev) |

RunnerBar uniquely sits in *all three*. No competitor in the data does. That intersection is the brand opportunity.

---

## 2 · Per-competitor analysis

### 2.1 [Raycast](https://www.raycast.com) — the macOS launcher gold standard
- **Tagline:** "Your shortcut to everything."
- **Palette:** Deep charcoal `#0C0C0C` · pure white `#FFFFFF` · signature red-pink accent `#FF6363`
- **What to steal:** the *keyboard-shortcut chip* visual language, dark-mode + signature accent gradient combo.

### 2.2 [Warp](https://www.warp.dev) — terminal-as-platform
- **Tagline:** "Ship better software with any agent."
- **What to steal:** the *inline copy-install command as primary CTA*. Vocabulary of "agentic."

### 2.3 [Linear](https://linear.app) — the dark-mode benchmark
- **Tagline:** "The product development system for teams and agents."
- **What to steal:** **the structural restraint** — numbered sections, single accent color against black, mono for IDs.

### 2.4 [CleanShot X](https://cleanshot.com) — Mac utility commercial benchmark
- **Tagline:** "Capture your Mac's screen like a pro."
- **What to steal:** the *"floating UI on a desktop wallpaper"* hero shot.

### 2.5 [Depot](https://depot.dev) — CI performance positioning
- **Tagline:** "Build faster. Waste less time."
- **What to steal:** the **animated workflow visualization** is *exactly* what RunnerBar shows in its popover.

### 2.6 [Blacksmith](https://www.blacksmith.sh) — the loud GitHub Actions challenger
- **Tagline:** "The fastest way to run your GitHub Actions."
- **What to steal:** the **YAML-diff hero** and unapologetic single-accent maximalism.

### 2.7 [Cursor](https://cursor.com) — premium AI-tool aesthetic
- **Tagline:** "Built to make you extraordinarily productive, Cursor is the best coding agent."
- **Palette:** Off-white / cream `#F9F9F7` · deep black · soft greys.
